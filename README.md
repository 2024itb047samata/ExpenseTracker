# Student Expense Tracker

Student Expense Tracker is a Python command-line application that helps students manage their daily expenses efficiently. Users can add expenses, view expense history, calculate total spending, and identify the category with the highest expenditure. The application also includes a monthly budget feature that warns users when they exceed their spending limit. It is lightweight, works completely offline, and uses Python lists and dictionaries for data storage. This project demonstrates fundamental Python concepts such as functions, loops, conditionals, and data structures.
